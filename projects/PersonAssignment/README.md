# Person Assignment

### Create a Person Object

1. 3 Variables
    - First Name
    - Last Name
    - Date of Birth (LocalDate Object)

2. Two Constructors
    - One with no parms
    - One with with 3 parms (FirstName, LastName, Date Of Birth)
    
3. Getters & Setters
    - Getter and Setter for each Variable
    
4. Methods
    - Custom toString
    - Custom equals
    - Custom compareTo
